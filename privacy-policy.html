Políticas de privacidad y declaración de cookies

Su privacidad es vital para Game room. Esta declaración de privacidad proporciona información sobre la información personal que recopila Game room y cómo Game room utiliza esa información personal.

Recopilación de información personal

Game room puede recopilar y usar los siguientes datos personales:

  • Información que proporciona en el proceso de registro en el sitio web;
  • Detalles sobre su uso del sitio web gameroom.com;
  • Los datos de transacciones realizados a través de este sitio web;
  • Información que proporciona para suscribirse a los servicios de gameroom.com;
  • Otra información que envíe a Game room por su cuenta.

Usando información personal

Game room puede usar su información personal para:

  • Administrar este sitio web;
  • Personalizar el sitio web de acuerdo con su configuración;
  • Darle acceso a los servicios de gameroom.com;
  • Enviarle los productos que ha comprado;
  • Enviarle declaraciones y facturas;
  • Cobrar pagos de usted;
  • Enviarle mensajes de comunicaciones de marketing.

Cuando Game room divulga cualquier información a terceros ( agentes o subcontratistas ) para los fines mencionados anteriormente, los terceros están obligados a utilizar esa información personal bajo los requisitos de esta declaración de privacidad.

Además de las divulgaciones razonablemente necesarias para los fines identificados anteriormente, Game room puede divulgar su información personal en la medida requerida por la ley, en caso de procedimientos legales o procedimientos legales prospectivos, y para establecer, ejercer o defender sus derechos legales.

Controles, opciones y derechos de los usuarios.

Game room puede ofrecerle ciertos controles y opciones con respecto a la información recopilada, cómo se usa la información y cómo se comparte.

Estos controles y opciones pueden incluir la capacidad de actualizar, corregir o eliminar la información que nos ha proporcionado o la información que Game room ha recopilado mediante el uso de nuestros servicios. También pueden incluir la capacidad de optar por no recibir notificaciones, promociones, ofertas u otra publicidad de Game room.

Dependiendo de su lugar de residencia, puede tener el derecho, de conformidad con la ley aplicable, de solicitar acceso, eliminación y portabilidad de su información. Para obtener información sobre cómo enviar dicha solicitud, comuníquese support@gameroom.com.

Game room no lo discriminará por ejercer sus derechos y opciones, aunque algunas de las funciones y características disponibles en nuestros servicios pueden cambiar o ya no estar disponibles para usted.

Asegurando sus datos

Game room toma todas las precauciones técnicas y organizativas razonables para evitar la pérdida, mal uso o alteración de su información personal.

Game room almacena toda la información personal que proporciona en sus servidores seguros. La información relacionada con las transacciones electrónicas realizadas a través de gameroom.com está protegida por tecnología de cifrado.

Transferencias de datos transfronterizas

La información que recopila Game room puede almacenarse, procesarse y transferirse entre cualquier país en el que opera Game room. Es necesario permitir el uso de la información que sigue a esta política de privacidad.

Se requiere su acuerdo para tales transferencias transfronterizas de información personal.

Actualizando esta declaración

Game room puede actualizar esta política de privacidad publicando una nueva versión en este sitio web. Se recomienda consultar esta página ocasionalmente para asegurarse de que está familiarizado con cualquier cambio.

Otros sitios web

Este sitio web contiene enlaces a otros sitios web. Game room no es responsable de las políticas o prácticas de privacidad de ningún tercero.

Contactar con Game room

Si tiene alguna pregunta sobre esta política de privacidad o el tratamiento de Game room de su información personal, contáctenos por correo electrónico a support@gameroom.com
